## Weiteres
Die App nutzt Bootstrap als CSS-Framework. Bootstrap wurde installiert mit

```bash
npm install boostrap@5.3.3
```

Die `.scss` files werden zu `.css` konvertiert via VS-Code-Extension `Live Sass Compiler v6.1.2`
